# 🧮 Projeto Calculadora Web

Este é um projeto de uma calculadora digital com design moderno e interface fluida, desenvolvido como parte dos meus estudos em Análise e Desenvolvimento de Sistemas.
🌐 **Acesse o projeto publicado aqui:** https://k3117.github.io/calculadora/

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação dos elementos da calculadora.
- **CSS3**: Estilização visual, layout responsivo com Flexbox/Grid e efeitos modernos de desfoque (Glassmorphism).
- **JavaScript**: Lógica de programação para captura de eventos, operações matemáticas e manipulação do visor.

## 🛠️ Funcionalidades
- Operações matemáticas básicas (Soma, Subtração, Multiplicação e Divisão).
- Função de apagar o último caractere digitado (Backspace).
- Visor centralizado e ampliado para melhor legibilidade.
